# Ritter-Kasse

Eine kleine Web-App zur Verwaltung gemeinsamer Unternehmungen und Ausgaben.

## Aktuelle Version

**3.2.1**  
**Build:** `20260630-005`

## Wichtig zum lokalen Testen

Ab Version 3.x besteht die App aus mehreren Dateien.  
Bitte immer die komplette ZIP-Datei entpacken und die Dateien gemeinsam in einem Ordner lassen.

Nicht nur `index.html` alleine verschieben oder öffnen.

## Projektstruktur

```text
/
├── index.html
├── style.css
├── app.js
├── manifest.json
├── service-worker.js
├── version.json
├── icons/
│   ├── icon-192.png
│   ├── icon-512.png
│   └── apple-touch-icon.png
├── README.md
└── CHANGELOG.md
```

## Veröffentlichung über GitHub Pages

Alle Dateien aus dem entpackten Ordner müssen ins Repository hochgeladen werden.
